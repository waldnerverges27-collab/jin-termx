# PSQL Format

PostgreSQL query formatter

**Package:** psqlformat  
**Author:** JinDev  
**Repository:** https://github.com/waldnerverges27-collab/jin-termx  
**Official:** https://github.com/long2ice/psqlformat  
**Type:** Node.js global module (npm)  
**License:** MIT

## Description

PSQL Format is a SQL formatter specifically designed for PostgreSQL queries. It formats your SQL with consistent indentation, capitalization, and spacing, making complex queries more readable and maintainable.

## Dependencies

- Node.js LTS (nodejs-lts)

## Install

```bash
jinx install npm --psqlformat
```

## Uninstall

```bash
jinx uninstall npm --psqlformat
```

## Update

```bash
jinx update npm --psqlformat
```

## Notes

- Command: `psqlformat`
- Supports PostgreSQL-specific syntax
- Can be integrated into editors and CI pipelines
