# Zsh-autosuggestions

Fish-like autosuggestions for ZSH

**Package:** zsh-autosuggestions  
**Author:** DevCoreX  
**Repository:** https://github.com/waldnerverges27-collab/jin-termx  
**Official:** https://github.com/zsh-users/zsh-autosuggestions  
**Type:** ZSH plugin (git clone)  
**License:** MIT

## Description

Zsh-autosuggestions suggests commands as you type based on your history and completions, similar to the Fish shell. Suggestions appear as grey text after the cursor and can be accepted with the right arrow key.

## Dependencies

- ZSH, git, zoxide

## Install

```bash
jinx install shell --zsh-autosuggestions
```

## Uninstall

```bash
jinx uninstall shell --zsh-autosuggestions
```

## Update

```bash
jinx update shell --zsh-autosuggestions
```

## Notes

- Installed in `~/.zsh-plugins/`
- Suggests from history and completions
- Highly configurable color and behavior
