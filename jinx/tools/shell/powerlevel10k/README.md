# Powerlevel10k

Modern and fast ZSH theme with customizable prompts

**Package:** powerlevel10k  
**Author:** DevCoreX  
**Repository:** https://github.com/waldnerverges27-collab/jin-termx  
**Official:** https://github.com/romkatv/powerlevel10k  
**Type:** ZSH plugin (git clone)  
**License:** MIT

## Description

Powerlevel10k is a fast and feature-rich ZSH theme that provides a highly customizable prompt with segments for git status, command execution time, directory, and much more. It is designed for speed and flexibility with instant prompt option.

## Dependencies

- ZSH, git, zoxide

## Install

```bash
jinx install shell --powerlevel10k
```

## Uninstall

```bash
jinx uninstall shell --powerlevel10k
```

## Update

```bash
jinx update shell --powerlevel10k
```

## Notes

- Installed as a ZSH plugin in `~/.zsh-plugins/`
- Configured via `p10k configure` wizard
- Instant prompt for near-zero startup time
