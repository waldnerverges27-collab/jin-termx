# Banner Configuration for Termux

Customize Termux with Jin-TermX Banner

**Package:** jin-termx (banner config)  
**Author:** DevCoreX  
**Repository:** https://github.com/waldnerverges27-collab/jin-termx  
**Type:** Termux UI customization  
**License:** MIT

## Description

Configure a custom banner for Termux using Jin-TermX. This package allows you to set a personalized welcome message that appears when you start Termux, enhancing your terminal experience.

## Dependencies

- Termux (base installation)

## Install

```bash
jinx install ui --banner
```

## Uninstall

```bash
jinx uninstall ui --banner
```

## Update

```bash
jinx update ui --banner
```

## Notes

- Source file: `~/.local/share/jin-termx/assets/banner/devcorex.txt`
- removes Termux welcome message: motd
- Restart Termux to apply changes
